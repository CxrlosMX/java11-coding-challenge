# 🧩 Java 11 Code Challenge

Repositorio con la solución a un challenge técnico backend desarrollado en **Java 11**
---

## 🚀 Tecnologías utilizadas

- Java 11
- Programación orientada a objetos
- API moderna de `String`
---

## 📌 Descripción general

El proyecto contiene la solución de 3 tareas independientes:

1. **Task 1:** Migración/refactorización de métodos a Java 11.
2. **Task 2:** Impresión de números en formato tabla.
3. **Task 3:** Construcción del cuadrado más grande posible usando dos palos.

Cada tarea se encuentra separada en su propio paquete para mantener una estructura clara y facilitar su revisión.
